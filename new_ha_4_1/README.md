# Аналіз заробітних плат (goit-algo-hw-04)

Цей проект — це модульна Python-програма для аналізу текстових файлів із даними про зарплати розробників. Програма обчислює загальну суму виплат та середню заробітну плату, автоматично ігноруючи дублікати та виправляючи помилки у форматі даних.

## Структура проекту

*   `main.py` — головний файл для запуску програми.
*   `clear_salary.py` — модуль для зчитування, очищення даних та видалення дублікатів.
*   `salary_calculation.py` — модуль для виконання математичних розрахунків.
*   `salary_file.txt` — текстовий файл із даними (створюється автоматично, якщо відсутній).

## Як налаштувати та запустити

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/pahan2024/goit-algo-hw-04
2.Підготуйте дані:
У папці з проектом знайдіть файл salary_file.txt (або запустіть програму, щоб вона його створила). Додайте дані у форматі:
text
Alex Korp,3000
Nikita Borisenko,2000
3.Запустіть програму:
Перейдіть у папку з проектом та запустіть main.py:
bash
python main.py

Особливості програми
Автоматизація шляхів: Завдяки модулю os, програма сама знаходить файл із даними, де б він не знаходився.
Безпека даних: Реалізовано видалення дублікатів за ім'ям співробітника.
Стійкість до помилок: Обробляються винятки FileNotFoundError та ValueError (якщо замість суми вказано текст).
